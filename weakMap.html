JavaScript WeakMap

A comprehensive guide to understanding and using WeakMap in JavaScript

What is WeakMap?

A WeakMap is a collection of key/value pairs where:

  • Keys must be objects (not primitive values)
  • Values can be any JavaScript type
  • Keys are weakly referenced (don't prevent garbage collection)
  • Not enumerable (can't loop over entries)
// Creating a WeakMap
const weakMap = new WeakMap();

// Keys must be objects
let obj1 = { name: "Object 1" };
let obj2 = { name: "Object 2" };

// Setting values
weakMap.set(obj1, "Value for Object 1");
weakMap.set(obj2, "Value for Object 2");

// Getting values
let value1 = weakMap.get(obj1);
console.log(value1); // "Value for Object 1"

WeakMap vs Map

Feature WeakMap Map
Key Types Objects only Any value
Iterability Not enumerable Enumerable
Size Property No Yes
Garbage Collection Keys can be collected Keys prevent collection
Performance Better for large datasets Good for general use

WeakMap Methods

WeakMap has only four methods:

  • set(key, value) - adds a key-value pair
  • get(key) - returns the value for a key
  • has(key) - checks if a key exists
  • delete(key) - removes a key-value pair
const wm = new WeakMap();
let key = {};

// set() - add a key-value pair
wm.set(key, "some value");

// has() - check if key exists
console.log(wm.has(key)); // true

// get() - retrieve value
console.log(wm.get(key)); // "some value"

// delete() - remove key-value pair
wm.delete(key);
console.log(wm.has(key)); // false

Common Use Cases

Private Data Storage

Store private data for objects without exposing it directly.

DOM Element Metadata

Associate metadata with DOM elements without modifying them.

Caching

Cache computed values that can be garbage collected when no longer needed.

// Example: Private data for objects
const privateData = new WeakMap();

class Person {
  constructor(name, age) {
    privateData.set(this, { name, age });
  }

  getName() {
    return privateData.get(this).name;
  }

  getAge() {
    return privateData.get(this).age;
  }
}

let person = new Person("Alice", 30);
console.log(person.getName()); // "Alice"
console.log(person.name); // undefined (private!)
